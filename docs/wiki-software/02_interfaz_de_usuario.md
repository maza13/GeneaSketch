# 02 - Interfaz de Usuario e Workspace

## Objetivo
Describir como se organiza el espacio de trabajo para editar y analizar un arbol.

## Componentes principales
- Lienzo central para navegar el grafo.
- Paneles laterales para detalle, edicion y herramientas.
- Menus y atajos para operaciones frecuentes.

## Buenas practicas
- Trabajar por focos (rama/persona) para evitar ruido visual.
- Confirmar contexto antes de aplicar cambios en lote.
- Usar zoom/pan y filtros para sesiones largas.
