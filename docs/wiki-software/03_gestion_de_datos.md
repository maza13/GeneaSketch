# 03 - Gestion de Datos y Archivos

## Objetivo
Explicar los flujos de apertura, guardado, importacion y exportacion.

## Flujos
- Abrir/guardar `.gsk` como formato principal.
- Importar GEDCOM para migracion/interoperabilidad.
- Exportar segun necesidad de intercambio.

## Riesgos comunes
- Mezclar archivos de versiones incompatibles sin revisar warnings.
- Ignorar mensajes de recovery o integridad en carga.
