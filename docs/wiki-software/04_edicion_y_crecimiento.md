# 04 - Edicion y Crecimiento del Arbol

## Objetivo
Guiar la edicion cotidiana de personas, relaciones y eventos.

## Operaciones de usuario
- Crear/editar personas.
- Vincular parent-child y uniones.
- Añadir claims y fuentes con trazabilidad.

## Recomendaciones
- Mantener una claim preferida por predicado.
- Usar fuentes/citas para claims revisadas o verificadas.
- Registrar cambios graduales para facilitar auditoria.
