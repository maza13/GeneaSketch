# 07 - Inteligencia Artificial y AI Web Bridge

## Objetivo
Documentar como usar IA para proponer cambios sin perder control editorial.

## Flujo recomendado
1. Preparar contexto acotado y claro.
2. Generar propuestas.
3. Revisar item por item antes de aplicar.
4. Confirmar cambios con evidencia cuando aplique.

## Buenas practicas
- No aplicar lotes ambiguos sin revision.
- Tratar propuestas IA como borrador, no como verdad final.
