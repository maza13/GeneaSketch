# Glosario del Usuario

## Objetivo
Definir terminos frecuentes para uso de la aplicacion.

- **Arbol:** vista grafica de relaciones genealogicas.
- **Claim:** dato afirmado sobre una entidad (nombre, fecha, lugar, etc.).
- **Fuente:** evidencia asociada a una afirmacion.
- **Journal:** historial auditable de cambios.
- **Quarantine:** datos GEDCOM no mapeados preservados para no perder informacion.
- **Snapshot:** estado persistido del grafo en `graph.json`.
