# 01 - Conceptos Fundamentales

## Objetivo
Explicar los principios base de GeneaSketch para uso diario.

## Pilares
- **Local-first:** tus datos se guardan localmente en `.gsk`.
- **Modelo en grafo:** personas, uniones, fuentes y eventos se conectan como red.
- **Visual-first:** el arbol es la interfaz principal de trabajo.
- **Auditoria:** cada cambio queda registrado para trazabilidad.
