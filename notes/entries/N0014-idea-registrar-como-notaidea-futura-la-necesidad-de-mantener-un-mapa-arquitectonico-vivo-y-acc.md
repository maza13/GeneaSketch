# Mapa arquitectonico vivo por etapas para la separacion guiada

## Context

El diagnostico de separacion arquitectonica ya dejo un mapa maestro util, pero ese artefacto representa el estado previo al hard cut guiado. Si el proyecto entra despues a la fase de separacion, trabajar solo con el mapa diagnostico inicial volveria a introducir ceguera operativa.

Hace falta conservar esta idea como seguimiento futuro: mantener un mapa arquitectonico vivo y accesible durante cada etapa de la separacion para reflejar el estado real de fronteras, conexiones y hotspots despues de cada fase relevante.

## Insight

El valor no esta solo en tener "un mapa", sino en convertirlo en una referencia operativa por etapas:

- mapa diagnostico inicial para arrancar
- mapa actualizado al cierre de cada fase del plan guiado
- mapa consolidado final cuando la separacion haya madurado

Eso evita que la futura integracion en wiki se haga con informacion obsoleta y permite usar el mapa como instrumento de control arquitectonico durante la ejecucion, no solo como documentacion historica.

## Proposed Actions

- Retomar esta idea cuando se reactive el plan derivado del diagnostico y definir el formato del mapa vivo por fase.
- Vincular cada etapa futura del hard cut con una actualizacion del mapa operativo correspondiente.
- Integrar el mapa consolidado final a la wiki del software o a otro punto de consulta permanente solo cuando el estado resultante sea el vigente.
- Mantener esta nota enlazada con el plan de reentrada y con el paquete diagnostico base.
